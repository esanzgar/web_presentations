Forewords

  • Reusable user interface widgets/components/web applications
  • Encapsulation and interoperability
  • Introduced for first time in 2011
  • Standards being added by the W3C to HTML and DOM specifications
    • Web Components is still a very new technology, its specifications being evolved as browser implementers and Web developers gather experience using it.
